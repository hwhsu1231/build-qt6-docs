Obsolete Members for QStateMachine

The following members of class QStateMachine are deprecated. They are provided to keep old source code working. We strongly advise against using them in new code.

[explicit] QStateMachine::QStateMachine(QState::ChildMode childMode, QObject *parent = nullptr)

This function is deprecated. We strongly advise against using it in new code.

Constructs a new state machine with the given childMode and parent.

Warning: Do not set the childMode to anything else than ExclusiveStates, otherwise the state machine is invalid, and might work incorrectly.
