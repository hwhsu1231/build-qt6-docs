Marked (WebEngine Recipe Browser example), version 0.4.0

A full-featured markdown parser and compiler, written in JavaScript. Built for speed.

Marked is used in the WebEngine Recipe Browser example

The sources can be found in qtwebengine/examples/webenginewidgets/recipebrowser/assets/3rdparty/marked.js.
